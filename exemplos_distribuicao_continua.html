Modelos Probabilísticos Contínuos

Distribuição Uniforme - UD(c, d)

Exemplo: A densidade de probabilidade para x = 3 em uma uniforme U(2,6):

c <- 2
d <- 6
x <- 3
dunif(x = x, min = c, max = d)
[1] 0.25

A probabilidade acumulada P(X ≤ 3.5):

punif(q = 3.5, min = c, max = d)
[1] 0.375

O quantil x100q que satisfaz P(X ≤ x100q) = 0.1:

qunif(p = 0.1, min = c, max = d)
[1] 2.4

Uma amostra de 9 números aleatórios:

runif(n = 9, min = c, max = d)
[1] 3.121249 3.416955 5.741432 3.695714 3.994151 2.881458 5.061781 3.948886
[9] 4.150997

Gerando visualização de dados do exemplo:

# Parâmetros
a <- 2
b <- 6
x0 <- 3.5
h <- 1 / (b - a)

# Gráfico base
plot(
  NA,
  xlim = c(1.8, 6.2),
  ylim = c(0, 0.4),
  xlab = "X",
  ylab = "p(x)",
  main = "Densidade de Probabilidade – Uniforme U(2,6)",
  axes = FALSE
)

# Eixos
axis(1, at = c(a, x0, b))
axis(2)
box()

# Reta da densidade
segments(a, h, b, h, lwd = 2)

# Linha vertical no fim do suporte
segments(b, 0, b, h, lwd = 2)

# Área sombreada P(X ≤ 3.5)
polygon(
  x = c(a, x0, x0, a),
  y = c(0, 0, h, h),
  density = 20,
  angle = 45,
  border = "black"
)

Interpretação apartir da visualização dos dados:

A probabilidade de um número ser escolhido aleatóriamente dentro do intervalo é o mesmo para todos, e para valores fora do intervalo a probabilidade é zero.

Distribuição Beta - Beta(a, B)

Exemplo: A densidade de probabilidade para x = 0.2 em uma Beta(a = 3, B = 10):

a <- 3
B <- 10
x <- 0.2
dbeta(x = x, shape1 = a, shape2 = B)
[1] 3.543348

A probabilidade acumulada P(X ≤ 0.2):

pbeta(q = 0.2, shape1 = a, shape2 = B)
[1] 0.4416543

O quantil x100q que satisfaz P(X ≤ x100q) = 0.15:

qbeta(p = 0.15, shape1 = a, shape2 = B)
[1] 0.1143327

Uma amostra de 9 números aleatórios:

rbeta(n = 9, shape1 = a, shape2 = B)
[1] 0.3883399 0.2159298 0.3877575 0.1719027 0.2242423 0.1510470 0.1696621
[8] 0.2351519 0.1653531

Gerando visualização de dados do exemplo:

x0 <- 0.2

# Sequência de x
x <- seq(0, 1, length.out = 1000)
y <- dbeta(x, a, B)

# Gráfico base
plot(
  x, y,
  type = "l",
  lwd = 2,
  xlim = c(0, 1),
  ylim = c(0, max(y) * 1.1),
  xlab = "X",
  ylab = "p(x)",
  main = "Densidade de Probabilidade – Beta(3,10)",
  axes = FALSE
)

# Eixos
axis(1, at = c(0, x0, 1))
axis(2)
box()

# Área sombreada P(X ≤ 0.2)
x_fill <- seq(0, x0, length.out = 500)
y_fill <- dbeta(x_fill, a, B)

polygon(
  c(0, x_fill, x0),
  c(0, y_fill, 0),
  density = 20,
  angle = 45,
  border = "black"
)

Interpretação apartir da visualização dos dados:

Devido ao Beta alto, maior que o Alpha, é possível notar como ela é uma distribuição assimétrica à esquerda, com maior concentração perto de 0 e cauda mais longa à direita.

Distribuição Beta Generalizada - BetaG(a, B, c, d)

Enquanto a distribuição beta tem um intervalo fixo de [0, 1] a distribuição beta generalizada possuí um intervalo arbitrário [c, d]. Ou seja, é a mesma distribuição, porém com um intervalo diferente de [0, 1].

Distribuição Exponencial - Exp(B)

Exemplo: A densidade de probabilidade para x = 2 em uma Exp(B = 0.5):

x <- 2
B <- 0.5
dexp(x, B)
[1] 0.1839397

A probabilidade acumulada P(X ≤ 3):

pexp(3, B)
[1] 0.7768698

O quantil x100q que satisfaz P(X ≤ x100q) = 0.35:

qexp(0.35, B)
[1] 0.8615658

Uma amostra de 7 números aleatórios:

rexp(7, B)
[1] 0.8957001 1.1585366 1.6552249 2.5298617 1.4642082 1.8011059 3.9583114

Gerando visualização de dados do exemplo:

  ylab = "p(x)",
  main = "Densidade de Probabilidade - Exp(B = 0.5)",
  axes = FALSE

# Eixos
# axis(1, at = seq(0, 20, 1))
axis(1, at = c(0, 3, 10, 20))
axis(2)
box()

# Área sombreada P(X ≤ 0.2)
x_fill <- seq(0, x0, length.out = 500)
y_fill <- dexp(x_fill, B)

polygon(
  c(0, x_fill, x0),
  c(0, y_fill, 0),
  density = 20,
  angle = 45,
  border = "black"
)

Interpretação apartir da visualização dos dados:

Diante da distribuição exponencial onde Beta é igual a 0.5, a densidade é maior em tempos curtos e diminuí conforme o tempo aumenta.

A distribuição normal de u = 3, e o = 1 mostra que se trata de uma distribuição simétrica, com pico localizado em sua média que é 3.
